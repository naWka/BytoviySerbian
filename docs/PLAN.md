Единственный источник «что делаем сейчас» на уровне этапов. Задачи с критериями приёмки — в [BACKLOG.md](BACKLOG.md). Изменился план → сначала обнови этот файл и бэклог, потом код.

## Текущий фокус

**ПИВОТ ПЛАТФОРМЫ 2026-07-04** (см. DECISIONS): Telegram-бот → мобильное приложение на **Expo (React Native)**. Причина: владельцу Telegram неудобен, хочет приложение на телефоне. **Концепция та же** — «второй слой» бытового диалога для живущих в Сербии. Питон-бот удалён (остаётся в истории git).

Что добавляем к концепции: (1) память обучения — интервальные повторения (SRS), приложение помнит, что запомнилось, а что нет; (2) сохранение любимых карточек/слов; (3) большой контент — 2000+ единиц; (4) задел под будущего AI-репетитора.

## Стек (v3)

Expo SDK 57 · React Native 0.86 · expo-router (файловая навигация, табы) · TypeScript · zustand + AsyncStorage (прогресс и сохранённое) · контент — JSON в `content/`. Без бэкенда, без авторизации — один пользователь.

## Целевая структура кода

```
app.json, package.json, tsconfig.json     # Expo-конфиг
src/
├── app/                    # экраны (expo-router, файловая маршрутизация)
│   ├── _layout.tsx         # корневой layout + провайдер темы
│   ├── (tabs)/             # нижние табы
│   │   ├── _layout.tsx
│   │   ├── index.tsx       # Ситуации (список сценариев)
│   │   ├── learn.tsx       # Учить — хаб: разобрать новые / повторить / прогресс (только слова)
│   │   ├── vocab.tsx       # Слова (колоды)
│   │   ├── saved.tsx       # Сохранённое
│   │   └── progress.tsx    # Прогресс
│   ├── scenario/[id].tsx   # карточки-подготовки сценария (флэшкарты)
│   ├── deck/[id].tsx       # колода слов (список + «знаю ✓»)
│   ├── review/[mode].tsx   # тренировка темы/сценария/сохранённого
│   ├── triage.tsx          # разбор новых слов: свайпы «знаю/учить/позже»
│   ├── session.tsx         # учебная SRS-сессия слов (переворот + оценки)
│   ├── later.tsx           # отложенные слова («позже»)
│   └── sos.tsx             # SOS-фразы
├── components/             # SwipeDeck, FlipWordCard, Confetti, Flashcard, списки…
├── constants/theme.ts      # цвета, отступы, шрифты
├── hooks/                  # тема/цветовая схема
├── lib/
│   ├── types.ts            # типы контента и прогресса
│   ├── srs.ts              # алгоритм интервальных повторений (SM-2-lite)
│   ├── store.ts            # zustand: прогресс, сохранённое, статистика
│   └── content.ts          # загрузка и индексирование контента
content/
├── scenarios/*.json        # сценарии (will_hear + your_phrases)
├── vocab/*.json            # колоды слов
├── sos.json                # SOS-фразы
└── index.ts                # сборка всего контента
```
