## Целевая структура кода

```
bot/
├── main.py            # точка входа: Dispatcher, роутеры, long polling
├── config.py          # чтение .env
├── content.py         # загрузка data/scenarios.json при старте
├── keyboards.py       # инлайн-клавиатуры, callback-данные
├── ai.py              # обёртка над Claude API
├── db.py              # SQLite: init + log_event()
└── handlers/
    ├── start.py       # /start, меню, «Мне срочно»
    ├── scenarios.py   # карточка сценария
    └── training.py    # ролевой диалог (FSM)
data/scenarios.json    # весь контент
```
