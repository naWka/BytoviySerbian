## Стек (v4 — PWA)

Expo SDK 57 · React Native 0.86 · expo-router · TypeScript · react-native-web (экспорт в web). Цель поставки — **PWA** (устанавливается с браузера на телефон, офлайн через service worker). Состояние — zustand; локально AsyncStorage/localStorage (офлайн-кэш), в облаке **Supabase** (auth email+пароль + таблица `profiles.state jsonb` с RLS) — источник правды на пользователя, синк debounce-апсертом. Контент — JSON в `content/` (в бандле). Хостинг — GitHub Pages, деплой — GitHub Actions. Бэкенд-сервера своего нет: браузер ходит в Supabase напрямую. См. DECISIONS (BS-22).

## Целевая структура кода

```
app.json, package.json, tsconfig.json     # Expo-конфиг
src/
├── app/                    # экраны (expo-router, файловая маршрутизация)
│   ├── _layout.tsx         # корневой layout + провайдер темы
│   ├── (tabs)/             # нижние табы (стартовая — «Учить»)
│   │   ├── _layout.tsx
│   │   ├── learn.tsx       # Учить — одна кнопка «Учить сейчас» + счётчики (BS-18); стартовый таб
│   │   ├── vocab.tsx       # Слова (колоды)
│   │   ├── saved.tsx       # Сохранённое
│   │   └── progress.tsx    # Прогресс
│   ├── deck/[id].tsx       # колода слов (список + «знаю ✓»)
│   ├── review/[mode].tsx   # тренировка темы/сохранённого
│   ├── session.tsx         # единое занятие (BS-18): один поток, активное вспоминание, 2 стороны
│   └── suspended.tsx       # убранные из учёбы слова (возврат)
├── components/             # FlipWordCard, Confetti, WordRow, списки…
├── constants/theme.ts      # цвета, отступы, шрифты
├── hooks/                  # тема/цветовая схема
├── lib/
│   ├── types.ts            # типы контента (слова/колоды) и прогресса
│   ├── srs.ts              # интервальные повторения: учебные шаги + SM-2-ядро, сторона карточки (BS-18)
│   ├── learn.ts            # единая очередь занятия + счётчики вкладки «Учить» (BS-18)
│   ├── store.ts            # zustand: прогресс, сохранённое, статистика (+ дневной лимит новых)
│   └── content.ts          # загрузка и индексирование контента
content/
├── vocab/*.json            # колоды слов
└── index.ts                # сборка всего контента
```
