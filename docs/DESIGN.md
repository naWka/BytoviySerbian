# DESIGN.md — система оформления

Токены живут в `src/constants/theme.ts`. Экранные компоненты берут цвета только через `useTheme()` / `useGradients()`. Примитивы — в `src/components/ui.tsx` (Screen, Txt, Button, Surface, ProgressBar, Ring, Pill, IconChip, SegmentedControl, EmptyState).

> Ревизия v2 (2026-07-04, задача «дизайн всё ещё унылый»): предыдущая «тёплая бумага» была технически на месте, но плоская — системный шрифт, одинаковые hairline-карточки, низкая энергия. Ревизия добавила характер: фирменный шрифт, глубину, тёплые градиенты и цвет в списках. Строгую формулировку «никаких теней/градиентов» из v1 сознательно отменили (см. DECISIONS).

## Палитра — «тёплая бумага + терракота + мёд»

Сцена: взрослый человек листает слова в очереди в управе днём (светлая) и на диване вечером (тёмная). Обе темы обязательны, переключение системное.

- Светлая: тёплый бумажный фон `#F6EFE4`, чернильный текст `#241C15`, акцент — терракота `#C0562B`, вторичный тёплый — мёд `#E0972A`.
- Тёмная: тёплый уголь `#14100C`, тёплый светлый текст, акцент — коралл `#E88B5A`, мёд `#EEAE4E`.
- Никаких чистых `#000`/`#fff` в фоне/тексте; нейтрали тонированы в тёплый оттенок. Тень тоже тёплая (коричневая `shadow`), не серая.
- Семантика сохранена: hear (фиолет), say (зелёный), sos (красный), 4 цвета оценок SRS.
- Стратегия — Restrained как пол, но с «геройскими» моментами: цвет и градиент включаются на действиях, кольцах прогресса, серии и финише — там, где нужен всплеск энергии, а не на каждой плашке.
- Списки тем/сценариев раскрашены ротацией акцентов (`categoryColors`), чтобы не быть монохромными; у каждой строки — цветной чип-иконка (`IconChip`), а не плоский значок.

## Градиенты

`Gradients[scheme]` (`useGradients()`): `primary` (терракота), `streak` (мёд→терракота, пламя серии), `hear`, `say`. Только для геройских поверхностей: главная кнопка «Учить», чип серии, экран финиша. Не для текста (`background-clip:text` запрещён) и не для фоновых плашек.

## Типографика

Шрифт **Nunito** (тёплый скруглённый гротеск, поддержка кириллицы и латиницы) — один на всё, вес несёт иерархию. Подключается в `src/app/_layout.tsx` через `useFonts`; экран загрузки ждёт шрифт. Веса: `black` (900) — сербские слова и заголовки-цифры; `extrabold` (800) — h2, лейблы, кнопки-градиент, таб-бар; `bold` (700) — h3, кнопки; `semibold`/`medium` — подписи; `regular` — текст. Произношение — моноширинным (`Mono`), это осознанный контраст. Обращаться к весам через `Font.*`, не через `fontWeight` (кастомный шрифт весом не переключается).

## Форма и глубина

Радиусы крупные (карточки 20–26, чипы/пилюли — pill). Рамки hairline + мягкая тёплая тень (`elevation(c.shadow, level)`), level 1 — карточки/строки/плитки, level 2 — «живые» карточки (флип). Тень даёт объём, без неё бумага читалась плоско. Карточки с цветным «поясом» сверху (тонированный header под тип карточки), а не с боковой цветной полосой (боковые полосы запрещены). Кольцо прогресса (`Ring`, react-native-svg) — для дневной цели и «выучено».

## Движение (reanimated)

- Флип карточки в занятии: rotateY с perspective, ~360 мс, ease-out.
- Смена карточек/экранных состояний: entering/exiting 200–300 мс, ease-out, без bounce.
- Нажатия: лёгкий scale (0.99) + opacity на карточках и кнопках — тактильный отклик.
- Праздник только на завершении (конфетти, один раз), haptics на оценке/действии.
- `useReducedMotion` → декоративное выключается, состояние меняется мгновенно.

## Копирайт

Русский, короткий, тёплый. Кнопки — глаголы («Учить», «Повторить»). Пустые состояния объясняют следующий шаг, не «здесь пусто». Без тире-эмдаш.

## Анти-референсы (из PRODUCT.md)

Не Duolingo (кислотный зелёный, мультяшные маскоты, геймификация ради неё). Не шаблонный SaaS (серо-синий, hero-метрики). Не сербский флаг как палитра. Энергию даём тёплым цветом, шрифтом и глубиной, а не громкостью.
