# DESIGN.md — система оформления

Токены живут в `src/constants/theme.ts`. Экранные компоненты берут цвета только через `useTheme()`.

## Палитра — «тёплая бумага + терракота»

Сцена: человек листает слова в очереди в управе днём (светлая) и на диване вечером (тёмная). Обе темы обязательны, переключение системное.

- Светлая: тёплый бумажный фон `#FAF6F0`, чернильный текст `#201B16`, акцент — терракота `#C05B2E`.
- Тёмная: тёплый уголь `#161210`, тёплый светлый текст, акцент — коралл `#E58B5B`.
- Никаких чистых `#000`/`#fff` в фоне/тексте; нейтрали тонированы в тёплый оттенок.
- Семантика сохранена: hear (фиолет), say (зелёный), sos (красный), 4 цвета оценок SRS.
- Стратегия — Restrained: акцент только на действиях, выборе и индикаторах.

## Типографика

Системный шрифт. Сербские слова — крупно и жирно (extrabold), произношение — моноширинным. Шкала плотная (продуктовый регистр), большие цифры прогресса — rounded (iOS).

## Форма

Радиусы крупные (карточки 20–28), рамки hairline, мягкие тени только у «живых» карточек (свайп-колода, флип). Никаких боковых цветных полос и градиентного текста.

## Движение (reanimated)

- Свайп-колода отбора: физика пружин, наклон карточки, штампы «ЗНАЮ/УЧИТЬ/ПОЗЖЕ», haptics на решении. Это герой-анимация.
- Флип карточки в сессии: rotateY с perspective, ~350 мс, ease-out.
- Смена карточек/экранных состояний: entering/exiting 200–300 мс, ease-out, без bounce.
- Праздник только на завершении (конфетти, один раз).
- `useReducedMotion` → декоративное (конфетти, штампы-повороты) выключается, состояние меняется мгновенно.

## Копирайт

Русский, короткий, тёплый. Кнопки — глаголы («Учить», «Повторить», «Разобрать»). Пустые состояния объясняют следующий шаг, не «здесь пусто».
