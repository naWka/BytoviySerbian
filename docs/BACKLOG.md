# BACKLOG — задачи

Правила:
- Задачи создаёт Product Manager из хотелок владельца. Разработчики сами задачи не придумывают.
- Статусы: `idea` (нужны ответы владельца) → `ready` (можно брать) → `in-progress` (укажи, кто взял) → `done` (+ дата).
- Критерии приёмки — только то, что владелец может проверить руками (в приложении).
- Нумерация сквозная: BS-N.

## BS-14: Постоянные ID у слов — чтобы прогресс не съезжал — `done` 2026-07-04 (ждёт проверки владельцем)

Сделано: id карточки выводится из самого текста сербской фразы (хеш), а не из позиции в файле. Правка/вставка/перестановка слов в JSON больше не сдвигает отметки. Прогресса у владельца ещё нет — миграция не потребовалась.

Как пользователь, хочу, чтобы моя отметка «знаю» держалась за конкретным словом и не сбивалась, когда контент темы пополняют или правят.

Причина: сейчас приложение опознаёт слово по его номеру по порядку в файле. Если слово вставить или убрать в середине темы — все слова ниже сдвигаются, и приложение решает, что «знаю» относится к другому слову. Прогресс тихо съезжает.

Объём: у каждого слова и реплики — свой постоянный ID прямо в контенте; прогресс SRS и «сохранённое» привязаны к нему. Разово перенести уже накопленный прогресс на новые ID, чтобы у владельца ничего не пропало. Техническая правка «под капотом», экран не меняется.

Критерии приёмки:
- [ ] Отмечаю несколько слов в теме как «знаю», закрываю-открываю приложение — отметки на тех же словах
- [ ] После обновления/пополнения темы мои отметки остаются на своих словах, не съезжают на соседние

## BS-15: Кнопка «знаю ✓» и сворачивание выученного вниз — `done` 2026-07-04 (ждёт проверки владельцем)

Сделано: раздел темы («Слова» → тема) стал списком слов; у каждого слова кнопка «знаю ✓» (помечает выученным сразу, без режима «Учить») и ★ сохранить; тап по слову раскрывает пример и «ложный друг». Выученные уходят в свёрнутую группу «Выучено · N» внизу, раскрывается тапом. «знаю» можно снять.

Как пользователь, хочу отметить слово «знаю» прямо при листании и чтобы выученное не мозолило глаза в списке темы.

Объём:
- На карточке слова при листании — кнопка «знаю ✓» (отмечает слово выученным сразу, без режима «Учить»).
- В списке слов темы выученные не показываются вперемешку: они собираются в свёрнутую группу «Выучено ▸» внизу. Тап по группе — раскрыть/свернуть. Сверху — слова, которые ещё учишь.
- Слово можно снять с «знаю» (вернуть в учёбу).

Критерии приёмки:
- [ ] На карточке слова жму «знаю ✓» — слово уходит из основного списка вниз, в «Выучено»
- [ ] Внизу темы вижу свёрнутую группу «Выучено (N)», раскрываю её тапом
- [ ] Могу снять отметку «знаю» — слово возвращается наверх
- [ ] Отметки не пропадают после перезапуска

## BS-16: Колода «Глаголы» со спряжением — `ready`

Как пользователь, хочу учить самые ходовые сербские глаголы и видеть, как они звучат «по лицам и родам» (я делаю / ты делаешь / он делал / она делала), с примерами, где и когда так говорят.

Причина: сейчас 28 колод слов, но отдельной колоды глаголов нет, и спряжения нигде не видно. Для живого разговора глаголы важнее всего — без них слова не собираются во фразу.

Объём:
- Новая колода «🔤 Глаголы» в разделе «Слова»: ~40 самых частых бытовых глаголов (быть, иметь, хотеть, мочь, идти, знать, говорить, делать, работать, жить, есть, пить, платить, купить, давать, брать, помогать, ждать, звонить, понимать и т. п.).
- На карточке глагола, кроме обычного (кириллица, латиница, произношение, перевод):
  - **настоящее время** по лицам: ja / ti / on-ona / mi / vi / oni;
  - **прошедшее время** с родами: мужской и женский (radio / radila), ед. и мн. число;
  - 1–2 примера употребления с переводом и коротким «когда так говорят»;
  - пометка «ложный друг», если есть.
- Как именно показать таблицу спряжения на карточке — на усмотрение разработчика (аккуратно, читается с одного взгляда); решение зафиксировать в DECISIONS.md. Черновик контента — AI + вычитка AI-корректором «как носитель», пометка «черновик, не проверено носителем-человеком».

Критерии приёмки:
- [ ] В разделе «Слова» появилась колода «Глаголы»
- [ ] Открываю глагол → вижу настоящее время по лицам (я / ты / он-она / мы / вы / они)
- [ ] Вижу прошедшее с родами (я делал / я делала)
- [ ] На карточке есть пример(ы) с переводом и подсказка, когда так говорят
- [ ] В колоде ~40 глаголов, карточка читается с одного взгляда

## BS-17: «Учить» — отбор слов «знаю / учить / позже» + сессии + живой дизайн — `done` 2026-07-04 (ждёт проверки владельцем)

Как пользователь, хочу заходить в «Учить», сначала отбирать слова (знаю / не знаю — учить / пока не интересно), потом учить отобранное с повторениями и видеть прогресс — как в популярных приложениях для слов. Сценарии из «Учить» убрать полностью.

Объём:
- Вкладка «Учить» — только слова из колод, без сценариев.
- **Разбор новых**: стопка карточек со свайпами (и кнопками): вправо «Знаю ✓», влево «Учить», вниз «Позже». «Позже» = не интересно сейчас, лежит в «Отложенных», можно вернуть. Пачки по 10 слов, темы перемешаны.
- **Учебная сессия**: карточка-переворот (тап — перевод), оценки «не помню / трудно / помню / легко» (интервальные повторения как раньше); «не помню» возвращает слово в конец этой же сессии.
- **Отложенные**: отдельный экран, слово можно вернуть в разбор или сразу взять в учёбу.
- Прогресс на вкладке: разобрать / повторить / учу / выучено, серия дней.
- Дизайн живее: тёплая палитра вместо серо-синей, физика свайпов, 3D-переворот, конфетти в конце сессии, лёгкая вибрация на действия.

Критерии приёмки:
- [ ] Открываю «Учить» → вижу «Разобрать новые» и «Повторить N» (если есть что повторять)
- [ ] В разборе свайпаю: вправо «знаю», влево «учить», вниз «позже» — и кнопки работают так же
- [ ] После пачки — итог (сколько куда) и кнопка «Учить сейчас»
- [ ] В сессии тапаю карточку — переворачивается на перевод; оцениваю — приходит следующая
- [ ] «Не помню» — слово вернётся ещё раз в этой же сессии
- [ ] В конце сессии — праздничный экран с итогами и серией дней
- [ ] «Отложенные» открываются, слово можно вернуть
- [ ] Всё переживает перезапуск приложения
