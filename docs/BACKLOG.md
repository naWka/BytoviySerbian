## BS-18: Переработка механики учёбы слов — один поток + активное вспоминание + говорение — `done` 2026-07-04 (ждёт проверки владельцем)

Сделано: вкладка «Учить» = одна кнопка «Учить сейчас» (число = повторить + новые сегодня) + строка «Новых сегодня: X из N» + счётчики «Учу / Выучено» + серия дней. Отдельный экран разбора «знаю/учить/позже» (triage) убран. Занятие — единый поток: просроченные + новые (в пределах дневного лимита 10/день), перемешанные по темам (round-robin по колодам). На каждой карточке активное вспоминание: вопрос → «Показать» → ответ → оценка (нельзя оценить, не открыв ответ). Две стороны выводятся из зрелости слова: новые/свежие/забытые — узнавание (серб→рус), зрелые (reps≥2 и интервал≥3 дней) — говорение (рус→вспомни серб.). Новые/забытые проходят короткие учебные шаги (≈1 мин / 10 мин) до выхода «на дни» (Anki learning steps); «Не помню» и незавершённые шаги возвращают слово в конец занятия. На карточке кнопка «Убрать» → слово вне учёбы; экран «Убранные» (бывш. «Отложенные») возвращает слово обратно. График: учебные шаги + SM-2-ядро; FSRS — задел на потом (см. DECISIONS). Дизайн BS-17 сохранён (переворот, конфетти, haptics, reduced motion).

Как пользователь, хочу учить слова так, как это работает в лучших приложениях: одна кнопка «Учить», где новые слова и повторение идут вперемешку; я сначала пытаюсь вспомнить сам, потом проверяю; а слова, которые уже неплохо знаю, тренирую «наоборот» — вспомнить, как сказать по-сербски. Нынешняя механика (разбор «знаю/учить/позже» отдельным шагом + отдельная сессия) не нравится.

Причина (фидбек владельца + методики эффективного запоминания):
1. «Знаю» сейчас ставится на глаз, без проверки — а *узнать* слово и *вспомнить его самому* это разное (testing effect нарушен).
2. Два отдельных шага (разбор → сессия) путают. В топ-приложениях один поток.
3. Новое слово сразу прыгает на «через день» — слишком большой первый шаг, забывается.
4. Нет дневного лимита новых → назавтра гора повторений, забрасываешь.
5. Тренируется только узнавание (серб → перевод), а цель — заговорить.

Решения владельца (2026-07-04): **один поток** (новые + повторение вперемешку, отдельный разбор убрать) · тренировать **узнавание + говорение**.

Объём:
- **Вкладка «Учить» → одна кнопка «Учить сейчас»** с числом «сколько сегодня» (пора повторить + сколько новых доступно). Серия дней и счётчики «учу / выучено» остаются. Пусто — «на сегодня всё».
- **Один поток занятия**: в очередь идут слова, которым пора повторить, + новые слова (не больше дневного лимита), перемешанные по темам. Отдельный экран «разбор знаю/учить/позже» убираем.
- **Активное вспоминание**: карточка показывает вопрос, ты пытаешься вспомнить, тапаешь «Показать» — видишь ответ, потом оцениваешь. Нельзя «знаю», не увидев ответа.
- **Две стороны**:
  - новые и «свежие» слова — узнавание: серб. слово → перевод;
  - слова, которые уже неплохо сидят, — говорение: русский → вспомни, как сказать по-сербски.
  Слово само «дорастает» от узнавания к говорению; если на говорении завалил — вернётся к коротким повторам.
- **Учебные шаги для новых**: новое/забытое слово сначала показывается 1–2 раза с короткими промежутками внутри занятия, и только потом уходит на дни. «Не помню» на любом слове — вернётся ещё раз в этом же занятии.
- **Дневной лимит новых слов** (по умолчанию ~10/день), чтобы не набрать гору повторений на завтра. На вкладке видно «новых сегодня: X из N».
- **«Убрать слово»**: на карточке можно убрать слово из учёбы (аналог прежнего «позже»); убранные — отдельным списком, можно вернуть.
- График повторений — умнее нынешнего (учебные шаги + настроенное SM-2-ядро; полноценный FSRS — задел на потом, см. DECISIONS). Оценки на карточке остаются.

Критерии приёмки:
- [ ] В «Учить» одна кнопка запуска; нет отдельного шага «разбор знаю/учить/позже»
- [ ] В занятии новые слова и повторение идут вперемешку
- [ ] На каждой карточке сначала вспоминаю сам, потом жму «Показать» и вижу ответ — только после этого оцениваю
- [ ] Часть карточек — «наоборот»: вижу русское, вспоминаю, как сказать по-сербски
- [ ] Новое слово не прыгает сразу на «через день»: сначала 1–2 быстрых показа в этом же занятии
- [ ] «Не помню» — слово вернётся ещё раз в этом же занятии
- [ ] Видно «новых сегодня: X из N»; когда лимит выбран — идут только повторения
- [ ] Слово можно убрать из учёбы и потом вернуть
- [ ] В конце занятия — итог и серия дней
