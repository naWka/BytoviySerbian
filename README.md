# Bytoviy Serbian

Мобильное приложение для русскоговорящих в Сербии: подготовка к бытовым разговорам («что услышишь» + «чем ответить»), тематические колоды слов и память обучения с интервальными повторениями.

Приложение на **Expo (React Native)** — ставится на телефон через Expo Go. Без авторизации, для личного пользования.

## Запуск на телефоне

1. Установи на телефон приложение **Expo Go** (App Store / Google Play).
2. На компьютере, один раз:
   ```bash
   npm install
   ```
3. Запусти сервер:
   ```bash
   npx expo start
   ```
4. Отсканируй QR-код камерой (iOS) или из Expo Go (Android). Приложение откроется на телефоне.

Компьютер и телефон должны быть в одной Wi-Fi сети. Если не подключается — запусти `npx expo start --tunnel`.

## Контент

Весь контент — JSON в `content/` (сценарии и колоды слов). Черновики сгенерированы и вычитаны AI, помечены как непроверенные носителем-человеком.

Пересобрать индекс контента после изменения файлов:
```bash
node scripts/build-content-index.mjs
```
