# Bytoviy Serbian (рабочее название)

Telegram-бот для русскоговорящих в Сербии: готовые фразы и тренировка диалогов для бытовых ситуаций — аптека, управа, рынок, такси и другие.

Статус: MVP в разработке. Проект разрабатывается полностью через AI-агентов.
