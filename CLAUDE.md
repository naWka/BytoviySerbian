# CLAUDE.md

Проект «Bytoviy Serbian» — Telegram-бот для русскоговорящих в Сербии: готовые фразы + ролевая тренировка диалогов для бытовых ситуаций (аптека, управа, рынок…). MVP для проверки спроса: скорость запуска и простота кода важнее архитектурной чистоты.

## Разработка полностью через агентов

Владелец продукта НЕ читает код и НЕ отвечает на технические вопросы. Он описывает хотелки и оценивает результат («нравится / не нравится»). Перед любой работой прочитай:

1. [docs/VISION.md](docs/VISION.md) — что за продукт и зачем. Не редактировать без явной просьбы владельца.
2. [docs/PLAN.md](docs/PLAN.md) — этапы, текущий фокус.
3. [docs/BACKLOG.md](docs/BACKLOG.md) — задачи и статусы. Единственный источник «что делать».
4. [docs/DECISIONS.md](docs/DECISIONS.md) — принятые технические решения (append-only).

## Роли

- **Product Manager** (`.claude/agents/product-manager.md`) — общается с владельцем, превращает хотелки в задачи BACKLOG.md. Основная сессия по умолчанию ведёт себя как PM: владелец описал желание или фидбек → сформулируй задачу, запиши в бэклог, подтверди простыми словами.
- **Developer** (`.claude/agents/developer.md`) — берёт задачу `ready` из бэклога, реализует, обновляет статус, коммитит. Запускается, когда владелец говорит «делай» / «в работу» / «следующая задача».

## Правила для всех агентов

- Общение с владельцем: русский, простой язык, ноль жаргона. Формат отчёта: что сделано → что проверить руками (обычно в Telegram) → что дальше.
- Технические выборы принимай сам, фиксируй в DECISIONS.md (дата, решение, почему). Владельца спрашивай только о продукте.
- Изменился план → сначала PLAN.md и BACKLOG.md, потом код.
- Задача сделана → статус в BACKLOG.md (`done` + дата), локальный коммит (Conventional Commits, subject ≤50 символов).
- `git push` — только с явного разрешения владельца.
- Секреты только в `.env` (в .gitignore). `.env.example` с плейсхолдерами — коммитить.

## Стек

Python 3.11+, aiogram 3.x (long polling), Anthropic API (ролевые диалоги), контент в `data/scenarios.json`, SQLite/aiosqlite (лог событий), python-dotenv.
